## Overview

Neon Mail reimagines the email client experience with a focus on speed, privacy, and intelligent organization. Built as a desktop application using Electron, it processes all data locally.

### Key Features

- **AI categorization** using local ML models
- **Scheduled sending** with timezone-aware delivery
- **Full-text search** with instant results
- **Offline support** with local SQLite storage
- **End-to-end encryption** for sensitive emails

### Design

The UI follows the same noir aesthetic as this portfolio, with a dark-first design and cool blue accents throughout.
